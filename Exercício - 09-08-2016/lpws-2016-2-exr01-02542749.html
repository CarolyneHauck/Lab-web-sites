01. O que é Internet?

A Internet é uma rede mundial de computadores, na qual possibilita a comunicação e troca de dados/informações entre os computadores e as demais redes existentes no mundo, através da tecnologia TCP/IP (Internet Protocol Suite).

02. O que é e quais são as funções de HTML, CSS e JavaScript?

O HTML é uma linguagem de programação utilizada para o desenvolvimento de páginas web, que informa aos navegadores como estruturar e apresentar o seu conteúdo em uma página.

O CSS (Cascading Style Sheets) é uma tecnologia/linguagem que possibilita aos desenvolvedores aplicar diversos estilos visuais à uma página, como por exemplo aplicar cores, animações, formatações...

O JavaScript é uma linguagem de programação por script que é interpretada e executada pelos navegadores (client-side) e que permite a interação entre uma página e os usuários sem a necessidade de requisição ao servidor web.

03. O que é um navegador para Internet?

O navegador é um programa responsável pela interpretação e visualização de páginas web.

04. O que é um plugin para navegador?

Os plugins foram desenvolvidos para compensar as limitações iniciais das páginas HTML. Eles atuam como software adicional nos navegadores, e tem o objetivo de processar dados específicos, como por exemplo, a reprodução de vídeos.

05. O que é uma extensão para navegador?

As extensões aprimoram os navegadores com recursos adicionais, fazendo com que fiquem mais robustos de acordo com as necessidades do usuário.

06. Como representar um endereço de uma página na Internet?

O URL é o responsável por representar o endereço de uma página ou aplicação web. Ele é composto por várias partes.

A primeira parte de um URL é denominada esquema, que é o protocolo que direciona a forma de conexão que o navegador deve realizar para acessar determinado conteúdo.

A segunda parte de um URL representa o nome do host em que o site está residido.

Os URLs podem conter uma parte adicional, que é denominada caminho, onde páginas adicionais de um determinado host podem ser acessadas.

07. O que é IP e DNS?

O endereço IP é composto por uma série de números que apontam à um determinado local (endereço) na rede.

Os navegadores são incapazes de reconhecer automaticamente cada endereço de IP existente. Eles precisam realizar uma consulta através de um sistema de nome de domínio (DNS).

O DNS é responsável por converter um URL em seu respectivo endereço IP, possibilitando assim o acesso às informações contidas no endereço desejado.

08. O que é phishing?

O Pishing é uma técnica maliciosa realizada por indivíduos com falsa identidade na internet com o objetivo de enganar e roubar informações pessoais dos usuários, onde normalmente são realizadas através de sites falsos.

09. O que é software livre e software proprietário?

Os softwares livres, ou de código aberto, são disponibilizados a todos para visualização/edição do código-fonte para que sejam desenvolvidos e aprimorados de forma aberta e cooperativa.

Ao contrário dos softwares livres, os softwares proprietários são privativos e licenciados com direitos exclusivos ao seu desenvolvedor.
